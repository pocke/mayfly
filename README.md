Mayfly
===========

Mayfly restart the command at regular intervals.

Installation
---------------

```sh
go get github.com/pocke/mayfly
```




Usage
----------

```sh
Usage of mayfly:
  -t int
    	time as second (default 600) (default 600)
```

```sh
# restart gulp every 60 seconds.
$ mayfly -t 60 gulp
```
